# 1. 데이터 설명
다양한 날씨환경, 주행상황에서 카메라를 이용하여 취득한 국내외 도로 주행 영상 데이터의 영상원본과 영상프레임별 속성정보.

국가의 자율주행 기술 발전 지원을 위해 자율주행 인식기술개발 업체, 연구소, 평가기관 등이 편리하게 활용할 수 있도록
공공데이터포털과 자동차데이터포털을 통해 개방합니다.

## 1) 데이터 개요
- 477건의 시나리오 기반 주행데이터
- 구성 요소:
  1. 주행 중 촬영된 영상 및 이미지(JPEG)
  2. 주행 궤적을 기록한 GPS 데이터(.dat)
  3. 이미지 내 객체 및 차선에 대한 Annotation 정보(JSON)

# 2. 데이터 활용 방법
본 데이터는 자율주행 기술 관련 업체, 연구소 등이 ADAS기능 개발을 위한 인식 알고리즘 학습 및 개발에 
유용하게 사용될 수 있습니다.

또한, 자율주행 시뮬레이터, Semantic Segmentation 기술을 활용한 자율주행 알고리즘 개발, 다양한 교통 관련 정보와 융합을 통해
다양한 차세대 교통관련 기술 개발 등에 활용 가능합니다.

### 1. 한국자동차연구원 국가중점데이터 개방 DB시스템 접속
- drivingdata.bigdata-car.kr

### 2. 회원가입 및 로그인

### 3. 시나리오 선택
- 메뉴 > 영상정보 > 시나리오별 영상정보 > 영상 선택

### 4. 자료 요청
- 이미지 자료 요청: 선택한 영상의 300frame 원본 이미지, 완성이미지(bounding box 표기 이미지)
- 정식자료 요청: 원본이미지, 완성이미지, CFG(DAT), 환경정보(CSV), GPS(DAT), IVN(DAT), LDWS(DAT), 객체정보(JSON)
